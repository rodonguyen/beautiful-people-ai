# Naval Ravikant Life Advisor AI Clone Project

This document adapts the original **AI Clone Builder** project plan to focus on replicating **Naval Ravikant**'s personality as a **Life Advisor**. Naval's style is calm, philosophical, concise, aphoristic, first-principles oriented, and wisdom-focused—blending Eastern philosophy, rational thinking, wealth creation, leverage, health, happiness, and inner peace.

The core idea remains: ingest Naval's content (tweets, podcast transcripts, *The Almanack of Naval Ravikant*, blog posts), extract/augment insights, and build a conversational AI that responds **as Naval** — thoughtful, non-judgmental, principle-based, often distilling complex ideas into clear, memorable statements.

We prioritize the **bare code + TUI** version for quick validation and testing, as requested.

## 1. Core Concept & Adapted MVP Scope
- **Use Case** — Create a "Naval clone" Life Advisor. Input Naval's quotes, threads, or excerpts. The AI responds to life questions (e.g., "How should I think about happiness?", "Should I chase money or meaning?", "How do I build leverage?") in Naval's voice: calm, reflective, principle-driven, with aphorisms and first-principles reasoning.
- **Key Personality Traits to Emulate** (from Naval's content):
  - Calm, peaceful tone — emphasizes inner peace over excitement.
  - Concise & aphoristic — short, punchy wisdom statements.
  - Philosophical yet practical — blends stoicism, Buddhism, capitalism, leverage.
  - First-principles thinking — breaks down to fundamentals.
  - Core themes: Health > Happiness > Wealth (importance reverse of pursuit order); desire causes suffering; leverage (code, media, capital, labor); specific knowledge; judgment; compound interest in habits/money/relationships.
  - Responses often reframe questions, reduce desire, promote clarity/meditation, avoid status games.
- **MVP Features**:
  - Manual paste of Naval content (quotes, threads, excerpts from *Almanack*).
  - Enhanced extraction + augmentation tailored to Naval's style (generalize to principles, infer mental models).
  - RAG-based clone that responds **as Naval**.
  - TUI (terminal menu) for input, creation, querying.
- **Exclusions** — No multi-clone yet; no auto-fetching (add later via RSS or manual copy from navalmanack.com or @naval tweets).

## 2. Tech Stack (Bare/TUI Version)
- **Language** — Python 3.10+.
- **AI/ML** — Hugging Face Transformers, LangChain (RAG), Sentence Transformers (embeddings), FAISS (vector store).
- **LLM** — Start with local/open (e.g., google/flan-t5-large or better like Mistral via HuggingFaceHub if API key). For better Naval-like reasoning, consider OpenAI/Grok API later.
- **Interface** — Pure Python with `input()` + optional `rich` for nice terminal formatting.
- **Storage** — In-memory dict for speed (validate idea first).
- **Install** — `pip install langchain transformers faiss-cpu sentence-transformers rich` (rich optional).

## 3. Insight Extraction & Augmentation (Naval-Tailored)
- **Basic Extraction** — Summarize, extract entities (concepts like "leverage", "specific knowledge", people/books).
- **Augmentation Step** (critical for Naval mimicry):
  - Infer core principle/mental model.
  - Generalize to timeless rule.
  - Rephrase in aphoristic, calm style.
  - Add related Naval themes (e.g., link to happiness = peace, desire = suffering, compound interest).
- **Example Input** — "Desire is a contract you make with yourself to be unhappy until you get what you want."
  - Augmented Output:
    - specific: Desire creates unhappiness until fulfilled.
    - causation: Attachment to outcomes → mental suffering.
    - generalized: "Peace comes from dropping unnecessary desires. Happiness is the absence of desire for what is not already present."
    - naval-style aphorism: "Desire is suffering in anticipation."

**Prompt for Augmentation** (tune this heavily):
```
From this Naval Ravikant statement or excerpt: {text}

1. Summarize the key insight concisely.
2. Infer the underlying causation or mental model.
3. Generalize to a broader life principle.
4. Rephrase as a calm, aphoristic Naval-style quote or rule.
5. Link to related Naval themes (health, happiness, wealth, leverage, judgment, peace, desire).
```

## 4. Code Structure
Single script for simplicity: `naval_clone_tui.py`

### 4.1 Imports & Globals
```python
from rich.console import Console
from rich.prompt import Prompt
import spacy
from transformers import pipeline
from langchain.embeddings import HuggingFaceEmbeddings
from langchain.vectorstores import FAISS
from langchain.llms import HuggingFaceHub  # or OpenAI
from langchain.chains import LLMChain, RetrievalQA
from langchain.prompts import PromptTemplate

console = Console()
nlp = spacy.load("en_core_web_sm")
summarizer = pipeline("summarization")
embeddings = HuggingFaceEmbeddings(model_name="sentence-transformers/all-MiniLM-L6-v2")
llm = HuggingFaceHub(repo_id="google/flan-t5-large")  # Upgrade for better results

clones = {}  # {name: [insights_dicts]}
```

### 4.2 Extraction + Augmentation
```python
augment_prompt = PromptTemplate(
    input_variables=["text"],
    template="""
From this Naval Ravikant statement or excerpt: {text}

1. Summarize the key insight concisely.
2. Infer the underlying causation or mental model.
3. Generalize to a broader life principle.
4. Rephrase as a calm, aphoristic Naval-style quote or rule.
5. Link to related Naval themes (health, happiness, wealth, leverage, judgment, peace, desire).
"""
)

augment_chain = LLMChain(llm=llm, prompt=augment_prompt)

def extract_and_augment(text):
    # Basic
    summary = summarizer(text, max_length=80, min_length=30, do_sample=False)[0]['summary_text']
    
    # Augment
    response = augment_chain.run(summary)
    parts = [p.strip() for p in response.split('\n') if p.strip()]
    
    insight = {
        "original": text,
        "summary": summary,
        "specific": parts[0] if len(parts) > 0 else "",
        "causation": parts[1] if len(parts) > 1 else "",
        "generalized": parts[2] if len(parts) > 2 else "",
        "aphorism": parts[3] if len(parts) > 3 else "",
        "themes": parts[4] if len(parts) > 4 else ""
    }
    return insight
```

### 4.3 Clone Building & Querying
```python
def build_naval_clone(insights):
    texts = [f"{i['aphorism']} | {i['generalized']} | {i['themes']}" for i in insights]
    vectorstore = FAISS.from_texts(texts, embeddings)
    return vectorstore

naval_response_prompt = """
Respond as Naval Ravikant: calm, philosophical, concise, using aphorisms and first-principles thinking. 
Focus on peace, leverage, specific knowledge, health, happiness over status or endless desire.
Question: {question}
Relevant wisdom: {context}
"""

def query_naval(vectorstore, question):
    qa_chain = RetrievalQA.from_chain_type(
        llm=llm,
        chain_type="stuff",
        retriever=vectorstore.as_retriever(search_kwargs={"k": 3}),
        chain_type_kwargs={"prompt": PromptTemplate.from_template(naval_response_prompt)}
    )
    return qa_chain.run({"question": question, "context": ""})  # context auto-filled
```

### 4.4 TUI Main Loop
```python
def get_multiline(prompt_text):
    console.print(prompt_text, style="bold cyan")
    lines = []
    while True:
        line = input()
        if line.strip().upper() == "END":
            break
        lines.append(line)
    return "\n".join(lines)

def main():
    while True:
        console.print("\n[bold green]Naval Ravikant Life Advisor Clone (TUI)[/bold green]")
        console.print("1 → Create/Load Naval Clone from content")
        console.print("2 → Ask Naval a life question")
        console.print("3 → Show stored insights")
        console.print("4 → Exit")
        choice = Prompt.ask("Choice", choices=["1","2","3","4"], default="4")

        if choice == "1":
            content = get_multiline("Paste Naval quote/thread/excerpt (end with END):")
            if not content.strip():
                continue
            insight = extract_and_augment(content)
            clones.setdefault("Naval", []).append(insight)
            console.print("[green]Insight added to Naval clone.[/green]")

        elif choice == "2":
            if "Naval" not in clones or not clones["Naval"]:
                console.print("[red]No insights yet. Add content first.[/red]")
                continue
            vectorstore = build_naval_clone(clones["Naval"])
            while True:
                question = Prompt.ask("Ask Naval (or 'back')")
                if question.lower() == "back":
                    break
                response = query_naval(vectorstore, question)
                console.print(f"\n[bold yellow]Naval:[/bold yellow] {response}\n")

        elif choice == "3":
            if "Naval" in clones:
                for i, ins in enumerate(clones["Naval"], 1):
                    console.print(f"{i}. {ins.get('aphorism', ins['summary'])}")
            else:
                console.print("[yellow]No insights stored.[/yellow]")

        elif choice == "4":
            console.print("[bold]Peace be with you... Exiting.[/bold]")
            break

if __name__ == "__main__":
    main()
```
